## 目标是一个分布式的免费,更通用广泛的开源网盘  
比起主流商业网盘cynet有哪些更好的地方  
1. 无需繁琐的认证信息，基于mastodon账号，申请简单使用广泛无门槛。  
2. 分布式的存储。cynet是通过mastodon媒体协议，使用隐写的方式把文件分割存储在mastodon实例上。  
3. 更安全，更隐私，更灵活的分享方式。
